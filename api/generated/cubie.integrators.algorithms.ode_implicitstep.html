Bases: BaseStepConfig

Configuration settings for implicit integration steps.

Parameters:

  • beta – Implicit integration coefficient applied to the stage derivative.

  • gamma – Implicit integration coefficient applied to the mass matrix product.

  • M (numpy.ndarray | sympy.matrices.dense.MutableDenseMatrix) – Mass matrix used when evaluating residuals and Jacobian actions.

  • preconditioner_order (int) – Order of the truncated Neumann preconditioner.

  • linsolve_tolerance – Linear solver tolerance used by the Krylov iteration.

  • max_linear_iters (int) – Maximum iterations permitted for the linear solver.

  • linear_correction_type (str) – Identifier controlling the linear correction operator.

  • nonlinear_tolerance – Convergence tolerance for the Newton iteration.

  • max_newton_iters (int) – Maximum iterations permitted for the Newton solver.

  • newton_damping – Damping factor applied within Newton updates.

  • newton_max_backtracks (int) – Maximum number of backtracking steps within Newton updates.
