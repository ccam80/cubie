cubie.integrators.loops.ode_loop

Outer integration loops for running CUDA-based ODE solvers.

The IVPLoop orchestrates an integration by coordinating device step functions, output collectors, and adaptive controllers. The loop owns buffer layout metadata and feeds the appropriate slices into each device call so that compiled kernels only need to focus on algorithmic updates.
