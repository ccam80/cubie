cubie.outputhandling

CUDA output management package for solver integrations.

The package centres on OutputFunctions, which compiles CUDA callables for saving time histories and summary metrics. Configuration helpers such as OutputConfig and the sizing utilities provide validated settings that shape buffer allocation and host-visible array layouts. The summarymetrics subpackage exposes summary_metrics and register_metric() so metric implementations can register their CUDA device callbacks during import.
