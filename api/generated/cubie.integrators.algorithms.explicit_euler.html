Initialise the explicit Euler step configuration.

Parameters:

  • dxdt_function – Device derivative function evaluating dx/dt.

  • observables_function – Device function computing system observables.

  • precision – Precision applied to device buffers.

  • n – Number of state entries advanced per step.

  • dt – Fixed step size.

  • driver_function – Optional device function evaluating spline drivers at arbitrary times.

  • solver_function_getter – Present for interface parity with implicit steps and ignored here.

  • numba_precision – Numba precision corresponding to the configured precision.

  • n – Dimension of the state vector.

  • fixed_step_size – Step size used for integration.
