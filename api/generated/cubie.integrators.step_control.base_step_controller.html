cubie.integrators.step_control.base_step_controller

Interfaces for step-size controller configuration and factories.

Notes

The abstract configuration and factory interfaces defined here encapsulate shared behaviour for fixed and adaptive step controllers. Concrete controllers extend these classes to compile CUDA device functions that implement specific control strategies.
