cubie.systemmodels.symbolic.cellml

Minimal CellML parsing helpers using cellmlmanip.

This wrapper is heavily inspired by chaste_codegen.model_with_conversions from the chaste-codegen project (MIT licence). Only a tiny subset required for basic model loading is implemented here.
