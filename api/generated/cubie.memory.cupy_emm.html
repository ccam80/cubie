cubie.memory.cupy_emm

CuPy External Memory Manager plugins for Numba CUDA contexts.

This module adapts CuPy synchronous and asynchronous memory pools for Numba’s External Memory Manager interface. It also exposes helpers that align CuPy’s current stream selection with streams generated by numba.cuda so that stream-ordered allocations behave consistently across both libraries.
