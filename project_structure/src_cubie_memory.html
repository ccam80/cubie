src/cubie/memory

The memory package coordinates GPU allocations across cubie. It exposes the package-level MemoryManager through default_memmgr so integrators can request array buffers, register CUDA streams, and plug in alternative allocators without reimplementing the control logic. Supporting modules describe allocation requests, track chunked response metadata, manage stream groups, and expose CuPy-backed External Memory Manager plugins for Numba contexts.

Entry point

default_memmgr instantiates MemoryManager with stream grouping and CuPy integration ready to configure. Typical callers obtain this singleton, register their instance identifier, and submit ArrayRequest objects that describe the arrays they need.

Public API

Core manager

cubie.memory.default_memmgr

Singleton interface coordinating GPU memory allocation and stream usage.

cubie.memory.mem_manager.MemoryManager(...)

Singleton interface coordinating GPU memory allocation and stream usage.

Array specifications

cubie.memory.array_requests.ArrayRequest(...)

Specification for requesting array allocation.

cubie.memory.array_requests.ArrayResponse([...])

Result of an array allocation containing buffers and chunking data.

Stream coordination

cubie.memory.stream_groups.StreamGroups([...])

Container for organizing instances into groups with shared streams.

CuPy External Memory Managers

cubie.memory.cupy_emm.current_cupy_stream(...)

Context manager that forwards a Numba stream into CuPy APIs.

cubie.memory.cupy_emm.CuPyAsyncNumbaManager(context)

Numba EMM plugin using CuPy MemoryAsyncPool for allocation and freeing.

cubie.memory.cupy_emm.CuPySyncNumbaManager(context)

Numba EMM plugin using CuPy MemoryPool for allocation and freeing.

Dependencies

The package requires numba.cuda for stream management and context access. CuPy is optional but enables the External Memory Manager plugins and stream interoperability helpers. These plugins are registered through MemoryManager when available and fall back to Numba’s native allocator otherwise.
