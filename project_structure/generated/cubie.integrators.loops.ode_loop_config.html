cubie.integrators.loops.ode_loop_config

Configuration helpers for CUDA-based integration loops.

The objects defined here capture shared and local buffer layouts alongside compile-critical metadata such as precision, save cadence, and device callbacks. They centralise validation so that loop factories receive consistent, ready-to-compile settings.
