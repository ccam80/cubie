  • _memory_type (str) – Type of memory allocation.

  • _unchunkable (tuple[str]) – Array names that cannot be chunked.

Notes

This container keeps attrs-managed metadata describing which arrays are chunkable and how dimensions map to batching axes so the array manager can transfer slices correctly.
