cubie.odesystems.symbolic.dxdt.generate_dxdt_fac_code(equations: Iterable[Tuple[Symbol, Expr]], index_map: IndexedBases | None = None, func_name: str = 'dxdt_factory', cse: bool = True) → str[source]

Emit Python source for a dx/dt CUDA factory.

Parameters:

  • equations – Iterable of dx/dt assignments expressed as (lhs, rhs) SymPy pairs.

  • index_map – Indexed bases that provide both symbol references and constants.

  • func_name – Name of the generated factory function.

  • cse – Whether to apply common subexpression elimination before emission.

Returns:

Python source code implementing the requested factory.

Return type:

str

Notes

The generated factory expects func(constants, precision) and returns a CUDA device function compiled with numba.cuda.jit().

cubie.odesystems.symbolic.dxdt.generate_dxdt_lines(equations: Iterable[Tuple[Symbol, Expr]], index_map: IndexedBases | None = None, cse: bool = True) → list[str][source]

Generate CUDA assignment statements for dx/dt updates.

Parameters:

  • equations – Iterable of (lhs, rhs) SymPy expressions describing dx/dt assignments.

  • index_map – Indexed bases that supply CUDA array references for each symbol.

  • cse – Whether to apply common subexpression elimination before emission.

Returns:

CUDA source lines that evaluate the dx/dt equations.

Return type:

list of str

Notes

index_map must expose all_arrayrefs containing each symbol in equations.
