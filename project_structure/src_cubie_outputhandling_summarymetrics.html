src/cubie/outputhandling/summarymetrics

The summarymetrics package houses the summary metric registry used by output handling to accumulate reductions during integration. Importing the package instantiates summary_metrics and eagerly imports the built-in metrics so that each registers its CUDA device update and save functions. External packages extend the system by decorating new metric classes with register_metric().

Public interface

summary_metrics

Registry and dispatcher for summary metrics.

register_metric

Create a decorator that registers a metric on instantiation.

Dependencies

  • Compiles device functions via cubie.CUDAFactory.CUDAFactory and numba.cuda.

  • Consumes save/update cadence configuration from cubie.outputhandling.
