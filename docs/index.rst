Welcome to cubie's documentation!
====================================

cubie is a Python library designed to provide an easy entry point for users to perform batch integration in parallel on NVIDIA GPUs.

The library includes a set of forward-simulating ODE solvers designed to integrate large batches of IVPs from different initial values or with different parameters.
